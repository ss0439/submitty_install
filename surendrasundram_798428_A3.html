    SQL Injection


    Source: https://testmatick.com/what-are-sql-injections-and-how-to-test-them/     		SQL Injection attacks trick the server into executing malicious SQL statements that can steal sensitive data, modify data, or disrupt operations.
    Learn more: OWASP SQL Injection
    Cross-Site Scripting (XSS)


    Source: https://www.linkedin.com/pulse/cross-site-scripting-xss-kamal-suryawanshi     		XSS attacks inject malicious scripts into web pages, which can steal user data, redirect users to phishing sites, or deface websites.
    Learn more: OWASP XSS Tutorial
    Cross-Site Request Forgery (CSRF)


    Source: https://securitysouls.com/a-tale-of-cross-site-request-forgery-csrf     		CSRF attacks trick a user's browser into performing unauthorized actions on a trusted website.
    Learn more: PortSwigger CSRF Guide
    Phishing


    Source: https://www.malwarebytes.com/blog/news/2018/09/6-sure-signs-someone-is-phishing-you-besides-email     		Phishing attacks attempt to trick users into revealing personal information or clicking on malicious links through deceptive emails or messages.
    Learn more: CISA Phishing Information
    Denial-of-Service


    Source: https://www.researchgate.net/figure/Denial-of-Service-DoS-attack-A-DDoS-Distributed-Denial-of-Service-attack-as-shown-in_fig2_325779181     		DoS attacks overwhelm a website or server with traffic, making it unavailable to legitimate users.
    Learn more: Cloudflare DoS Attack Guide
    Man-in-the-Middle (MitM) Attack


    Source: https://www.malwarebytes.com/blog/news/2018/07/when-three-isnt-a-crowd-man-in-the-middle-mitm-attacks-explained     		MitM attacks intercept communication between two parties, allowing attackers to steal data, eavesdrop on conversations, or manipulate traffic. This can happen on unsecured Wi-Fi networks or through compromised systems.
    Learn more: Tech Target
    Password Attacks


    Source: https://captex.bank/cybersecurity/password-attacks/     		Password attacks aim to guess or crack a user's password to gain unauthorized access to accounts. This can involve techniques like brute-force attacks (systematically trying different combinations), dictionary attacks (using common words and phrases), or social engineering to trick users into revealing their passwords.
    Learn more: Educative.IO
    Malware


    Source: https://www.kaspersky.com/resource-center/preemptive-safety/what-is-malware-and-how-to-protect-against-it     		Malware is malicious software that can infect devices and steal data, disrupt operations, or install additional harmful software. Malware can be disguised as legitimate applications or spread through phishing emails, malicious websites, or infected downloads.
    Learn more: Australian Signals Directorate
    Zero-Day Exploit


    Source: https://www.darkreading.com/threat-intelligence/zero-day-exploit-use-exploded-in-2021     		Zero-day exploits target vulnerabilities in software that are unknown to the vendor and for which no patch is yet available. These attacks are particularly dangerous because they can be highly successful before a fix is developed.
    Learn more: IBM
    Supply Chain Attacks


    Source: https://www.wallarm.com/what/what-is-a-supply-chain-attack     		Supply chain attacks target vulnerabilities in software development processes to inject malicious code into legitimate software. This can compromise a wide range of systems that rely on the affected software
    Learn more: Crowdstrike
